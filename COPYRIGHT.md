## Ownership guidance
In general, **if you created this source code yourself**, you normally hold the copyright automatically from the moment the code was created.

However, check these cases carefully:
1. **Employment / company work** → the company may own it if your contract says so.
2. **Freelance / client project** → ownership depends on the written agreement.
3. **Third-party libraries** → you own only your original code; external packages keep their own licenses.
4. **Copied code/assets** → do not claim ownership over copied material that you did not create.

## If you want to share it publicly

- `Apache-2.0` → permissive with patent terms
- `GPL-3.0` → strong copyleft
